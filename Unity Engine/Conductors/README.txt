To run the game:
  Open up the "Conductors_EXE" directory. 
  Run the file Conductors.exe

To Restart a Level:
  Press ESC.
  Clcik on the Restart button.

To Quit the Game:
  Press ESC to pause the game. 
  Click on the Quit button. (Note!: This will close the application and you would loose all progress)
